# 重生体验馆 | Identity Simulator

一款沉浸式文字体验游戏。你将随机投生为历史长河中某个真实存在过的普通人，在有限的选择中走完他们的一生。

## 体验地址

🔗 **https://nukyykun.github.io/identity-simulator/**

## 关于

这不是一个"赢"的游戏。没有最优解，没有隐藏结局，没有二周目奖励。

你能做的只有：接受你抽到的身份，在那个时代、那个身份的局限里，看看一个人的一生可以是什么样子。

## 特点

- **61个角色**，覆盖从古巴比伦到20世纪的普通人生
- **5种叙事基调**：炽烈(blaze)、漂泊(drift)、余烬(ember)、沉默(mute)、静水(still)
- **真实的历史约束**：每个角色的命运受限于其所处时代的社会结构
- **有限的选择**：分支不多，且大多数改变不了结局——正如真实人生
- **纯前端**：无需后端，打开即玩，支持离线

## 技术

纯静态前端项目（HTML + CSS + JS），角色数据存储在 `data/characters.json`。支持 PWA 离线访问。

## 本地运行

直接用浏览器打开 `index.html` 即可，或使用任意静态服务器：

```bash
npx serve .
```
