# 重生体验馆 · 人生色系规范 (Tone System)

## 设计原则

色系由角色的**人生情感基调**决定，而非其社会身份。
一个帝王的人生可以是「隐忍」的，一个乞丐的人生也可以是「燃尽」的。
分类依据是这个人经历了怎样的情感弧线，最终留给读者什么感受。

---

## 五种色调 (Tones)

### mute · 隐忍灰
- **情感弧线**: 被压制的沉默，忍而不发，内心丰富但一生被困
- **视觉感受**: 冷灰沉闷，像蒙尘的石碑
- **配色**:
  - 背景: `#0c0b0a`
  - 卡片/面板: `#1c1a18`
  - 强调色: `#6b635a` (暗灰褐)
  - 正文: `#a09890`
  - 高亮文字: `#d0c8c0`
- **适用人生**: 被命运裹挟无力反抗的、一生隐忍等待的、沉默的大多数

### drift · 漂泊青
- **情感弧线**: 无根的流离，一生在路上，归属感始终缺失
- **视觉感受**: 冷青湿润，像雨中的渡口
- **配色**:
  - 背景: `#080c0e`
  - 卡片/面板: `#141e22`
  - 强调色: `#5a8a8a` (黯青)
  - 正文: `#98b0ac`
  - 高亮文字: `#d0dcd8`
- **适用人生**: 背井离乡的、流亡漂泊的、始终在寻找但未找到的

### blaze · 燃尽赭
- **情感弧线**: 挣扎后的余烬，拼尽全力但最终燃尽
- **视觉感受**: 暗红温热，像熄灭的炭火
- **配色**:
  - 背景: `#0e0908`
  - 卡片/面板: `#241410`
  - 强调色: `#b05a3a` (赭红)
  - 正文: `#c0a898`
  - 高亮文字: `#e8d8cc`
- **适用人生**: 反抗者、战斗者、激烈燃烧后归于沉寂的

### ember · 微光琥珀
- **情感弧线**: 卑微中的创造，在困顿中依然有小小的光
- **视觉感受**: 暖黄柔和，像油灯下的手工纸
- **配色**:
  - 背景: `#0b0a08`
  - 卡片/面板: `#1e1a12`
  - 强调色: `#b89050` (琥珀)
  - 正文: `#b8a898`
  - 高亮文字: `#e8dcd0`
- **适用人生**: 默默创造的匠人、平凡中有温情的、小人物的坚韧

### still · 沉静靛
- **情感弧线**: 宿命的平静，看透了便不再挣扎
- **视觉感受**: 深靛沉稳，像夜海水面
- **配色**:
  - 背景: `#090a0e`
  - 卡片/面板: `#14161e`
  - 强调色: `#6878a0` (暗靛)
  - 正文: `#a0a8b8`
  - 高亮文字: `#d0d4dc`
- **适用人生**: 认命但不怨的、看破后归隐的、超然的旁观者

## 角色 tone 分配指南

给新角色分配 tone 时，问这三个问题：

1. **这个人一生的主情绪是什么？**
   - 压抑/克制 → mute
   - 漂泊/失根 → drift
   - 挣扎/激烈 → blaze
   - 温暖/卑微中的光 → ember
   - 平静/超然 → still

2. **读完这个人的故事，读者心里留下什么？**
   - 沉闷、压抑 → mute
   - 惆怅、怀念 → drift
   - 不甘、壮烈 → blaze
   - 温暖、感动 → ember
   - 释然、平和 → still

3. **如果用一个自然意象形容这个人的人生：**
   - 蒙尘的石碑 → mute
   - 雨中的渡口 → drift
   - 熄灭的炭火 → blaze
   - 油灯下的纸 → ember
   - 夜海水面 → still

## 技术实现

在 `characters.json` 中，每个角色包含 `"tone"` 字段：

```json
{
  "id": "xxx",
  "name": "某某",
  "tone": "mute",
  ...
}
```

前端通过 `data-tone` 属性切换 CSS 自定义属性：

```css
[data-tone="mute"] {
  --tone-bg: #0c0b0a;
  --tone-bg2: #1c1a18;
  --tone-accent: #6b635a;
  --tone-text: #a09890;
  --tone-bright: #d0c8c0;
}
```

JS 在进入人生/结局页时设置：

```javascript
const tone = char.tone || 'ember';
document.getElementById('screen-life').setAttribute('data-tone', tone);
document.getElementById('screen-ending').setAttribute('data-tone', tone);
```
