# 重生体验馆 · 人生色系规范 (Tone System)

## 设计原则

色系由角色的**人生情感基调**决定，而非其社会身份。
一个帝王的人生可以是「隐忍」的，一个乞丐的人生也可以是「燃尽」的。
分类依据是这个人经历了怎样的情感弧线，最终留给读者什么感受。

---

## 五种色调 (Tones)

### mute · 隐忍灰
- **情感弧线**: 被压制的沉默，忍而不发，内心丰富但一生被困
- **视觉感受**: 冷灰沉闷，像蒙尘的石碑
- **配色**:
  - 背景: `#0c0b0a`
  - 卡片/面板: `#1c1a18`
  - 强调色: `#6b635a` (暗灰褐)
  - 正文: `#a09890`
  - 高亮文字: `#d0c8c0`
- **适用人生**: 被命运裹挟无力反抗的、一生隐忍等待的、沉默的大多数

### drift · 漂泊青
- **情感弧线**: 无根的流离，一生在路上——但路的尽头也许有新的家
- **视觉感受**: 冷青湿润，像雨中的渡口
- **配色**:
  - 背景: `#080c0e`
  - 卡片/面板: `#141e22`
  - 强调色: `#5a8a8a` (黯青)
  - 正文: `#98b0ac`
  - 高亮文字: `#d0dcd8`
- **适用人生**: 背井离乡的、流亡漂泊的——无论最终是否找到归属

### blaze · 燃尽赭
- **情感弧线**: 挣扎后的余烬，拼尽全力——无论成败，活过了
- **视觉感受**: 暗红温热，像熄灭的炭火
- **配色**:
  - 背景: `#0e0908`
  - 卡片/面板: `#241410`
  - 强调色: `#b05a3a` (赭红)
  - 正文: `#c0a898`
  - 高亮文字: `#e8d8cc`
- **适用人生**: 反抗者、战斗者、激烈燃烧后归于沉寂的

### ember · 微光琥珀
- **情感弧线**: 卑微中的创造，在困顿中依然有小小的光
- **视觉感受**: 暖黄柔和，像油灯下的手工纸
- **配色**:
  - 背景: `#0b0a08`
  - 卡片/面板: `#1e1a12`
  - 强调色: `#b89050` (琥珀)
  - 正文: `#b8a898`
  - 高亮文字: `#e8dcd0`
- **适用人生**: 默默创造的匠人、平凡中有温情的、小人物的坚韧

### ease · 沉静靛
- **情感弧线**: 潇洒自在，看得透、放得下、活得明白
- **视觉感受**: 深靛沉稳，像夜海水面
- **配色**:
  - 背景: `#090a0e`
  - 卡片/面板: `#14161e`
  - 强调色: `#6878a0` (暗靛)
  - 正文: `#a0a8b8`
  - 高亮文字: `#d0d4dc`
- **适用人生**: 认命但不怨的、看破后归隐的、超然的旁观者

## Tone 情绪治理系统

### 核心规则

**tone 不仅是一个视觉标签，它治理角色的写作风格、情绪表达和分支结局走向。**

每个 tone 有一个**情绪重力值 (emotional gravity)**，决定了这个角色的文字可以"轻"到什么程度：

| tone | 情绪重力 | 情绪光谱 | 允许正面分支？ | 总体占比上限 |
|------|---------|---------|-------------|------------|
| mute | 高 | 压抑—隐忍—沉默 | 极少（最多一处微光） | ≤ 20% |
| drift | 中 | 惆怅—思乡—扎根—新生 | **允许且鼓励** | 不限 |
| blaze | 中 | 壮烈—不甘—激昂—痛快 | 允许（可以是壮烈的胜利） | 不限 |
| ember | 低 | 温暖—坚韧—满足—小确幸 | **默认正面** | 不限 |
| ease | 低 | 平静—释然—超然—自在 | **默认正面** | 不限 |

### 分层定义

**真·低情绪（Dark Tier）**: 只有 `mute`

mute 是唯一一个"写作中必须保持沉重基调"的 tone。mute 角色的人生底色是被困、被压制，即便有分支选择也很难改变命运走向。但即便是 mute，也允许一处"微光时刻"——只是这个微光会很快被现实盖灭。

**中性（Neutral Tier）**: `drift` + `blaze`

drift 和 blaze 角色的人生有苦有乐，不预设结局走向。drift 角色可以漂泊后落地生根，也可以一生无着；blaze 角色可以燃尽而亡，也可以拼出一片天。关键在于**尊重史实的前提下，不刻意渲染悲惨**。离开故土是悲，但新的地方也可以是家；挣扎激烈是苦，但拼搏本身可以带来成就感。

**暖调（Warm Tier）**: `ember` + `ease`

ember 和 ease 角色天然偏暖或偏静。ember 的写作要让读者感到温情和力量，哪怕是卑微的力量；ease 的写作要让读者感到潇洒、通透、自在——这个人从没觉得被困住过。这两个 tone 的分支结局默认是正面或中性的。

### 比例硬性约束

**在任何批次的角色集合中，mute（真·低情绪）角色占比不得超过 30%。**

当前推荐分布：
- mute: 15-25%（约4-8个/30人）
- drift: 15-25%
- blaze: 15-20%
- ember: 20-30%
- ease: 15-18%

### 第一性原则：人不知道自己会死

写每个角色的时候，记住：**这个人活着的时候不知道自己会死，不知道自己是"历史的尘埃"。** 他只知道今天太阳升起来了，明天还有事做。作者知道结局，但人物不知道。绝对禁止让"上帝视角的悲悯"渗透进人物的当下体验。

一个正在打仗的士兵想的是"下一仗怎么打"，不是"我好可怜我要死了"。一个在织机前站了十二小时的女工想的是"今天食堂有什么"，不是"我的人生好悲惨"。

### Tone = 主观乐观度的调节器

惨不惨是客观的（处境），觉不觉得惨是主观的（心态）。Tone 控制的是后者。五种 tone 的主观乐观度各不相同，而且乐观的**质感**也不同：

| tone | 主观乐观程度 | 质感 | 一句话 |
|------|-------------|------|--------|
| **blaze** | 高（轰轰烈烈） | 火 | 天塌下来当被盖，活得痛快 |
| **drift** | 高（随遇而安） | 风 | 吹到哪算哪，到哪都有新鲜事 |
| **ember** | 中偏高（坚忍微光） | 灯 | 日子再难，灯没灭过 |
| **ease** | 中性偏上（自在） | 水 | 本来就没觉得被困住 |
| **mute** | 低（但不是零） | 石 | 不说话，偶有苔藓 |

- **blaze / drift**：主观乐观程度**高**，但高法不同——blaze 是轰轰烈烈的热，drift 是轻盈好奇的风。两者共同点：他们自己不觉得苦。
- **ember**：主观乐观程度**中偏高**，是坚忍的微光。持灯人——日子可能苦，但心里有一盏灯（手艺/家人/信念/习惯），这盏灯没灭过。他的乐观不是 blaze 那种"管他呢老子痛快"，而是"日子再难，灯还亮着"。
- **ease**：主观上是**通透自在**的，是平静而潇洒的水面。看透了世事，活得比谁都明白，不被外物所累。
- **mute**：主观乐观程度**低**，是唯一允许"觉得苦"的类别。但即便如此，mute 的表达方式是沉默和克制，不是哭天抢地。

### Tone 对写作风格的具体治理

#### blaze 的写作规则
- 语调有力度、节奏快，像鼓点。**这个人活得带劲。**
- 他的主观体验是充实的、热的、有劲儿的——哪怕在打仗、在逃命、在拼命
- 允许壮烈也允许痛快——不是所有挣扎都以失败告终
- 允许冷幽默和自嘲（"老子拼了命，结果也就混了个温饱"也是一种blaze）
- 分支结局：可以是悲壮的，也可以是"虽然付出代价但赢了"
- summary 收尾落在"他做了什么"而非"他失去了什么"
- **禁止**：全程苦大仇深；禁止哀叹；禁止把所有选择都导向惨败；禁止以悲情收尾

#### drift 的写作规则
- 语调随遇而安，有好奇心。**漂泊对这类人来说是冒险而非苦难。**
- 允许且鼓励"他乡变故乡"的叙事——新的地方有新鲜的东西
- 写离别可以惆怅（一处即可），写新生活必须有烟火气和幽默
- 到了一个新地方，写他发现了什么有趣的事，而不是他多想家
- 分支结局：至少一条分支走向"安定/扎根/找到归属"
- **禁止**：每个事件都在渲染"无根"；禁止把异乡生活全写成苦难；禁止通篇愁苦

#### ember 的写作规则
- 语调温暖、从容，带一点烟火气
- 鼓励写日常中的小幸福、手艺带来的满足感、人与人之间的善意
- 允许困难但困难是衬底/背景板，不是主旋律——困难是一笔带过的，快乐是展开写的
- 分支结局：默认正面（安享晚年/技艺传承/家庭圆满），至多一条"命苦线"
- **禁止**：通篇强调卑微；禁止让"微光"变成"快灭的蜡烛"

#### ease 的写作规则
- 语调潇洒、通透，带一点看透世情的从容和幽默
- 这个人看得明白、活得自在，从不觉得自己被命运或身份困住
- 限制是真实的，但他在限制里活出了自己的活法——甚至是主动挑的路
- 允许冷幽默（看透了的那种豁达）
- 分支结局：默认平静而明亮的收尾（安然自得/活得透彻/超然自在）
- **禁止**：把"自在"写成"隐忍"或"认命"；禁止上帝视角的怜悯；禁止让人觉得这个人活得憋屈

#### mute 的写作规则
- 语调克制、冷静，像第三方在复述
- 不煽情但允许沉重——沉重来自「沉默」而非「悲情」
- 不是作者在替他哭，是他自己没说话
- 黑色幽默可以有，但是苦涩的那种
- 分支结局：大概率是不同形式的困顿/沉默，最多有一条"微光线"但不能逆天改命
- **禁止**：通篇绝望；禁止煽情；禁止俯视式悲悯（"多可怜啊"）；禁止让读者觉得"这个游戏好丧我不想玩了"

## 角色 tone 分配指南

给新角色分配 tone 时，问这三个问题：

1. **这个人一生的主情绪是什么？**
   - 压抑/克制 → mute
   - 漂泊/迁徙（无论结局好坏） → drift
   - 挣扎/激烈/战斗 → blaze
   - 温暖/卑微中的光 → ember
   - 潇洒/通透/超然 → ease

2. **读完这个人的故事，读者心里留下什么？**
   - 沉闷、压抑 → mute
   - 辽阔、感慨（可惆怅可温暖） → drift
   - 不甘、壮烈、痛快 → blaze
   - 温暖、感动 → ember
   - 释然、自在 → ease

3. **如果用一个自然意象形容这个人的人生：**
   - 蒙尘的石碑 → mute
   - 雨中的渡口（也许雨后有晴天） → drift
   - 熄灭的炭火（也许照亮过什么） → blaze
   - 油灯下的纸 → ember
   - 夜海水面 → ease

### 重要提醒

**分配 tone 时不要掉入"题材决定基调"的陷阱：**

- 下南洋的商人 ≠ 一定是 mute 或悲惨的 drift → 如果这人后来在海外发了财、成了当地富商，他可以是 drift（漂泊后扎根）甚至 ember（异国创业的满足）
- 战争中的人 ≠ 一定是 mute → 如果这人在战争中找到了自己的使命，可以是 blaze
- 宫女/奴隶 ≠ 一定是 mute → 如果这人最终被释放并过上了自己的生活，可以是 ember 或 drift

**先看这个人的人生全弧线和结局，再分配 tone。**

## 技术实现

在 `characters.json` 中，每个角色包含 `"tone"` 字段：

```json
{
  "id": "xxx",
  "name": "某某",
  "tone": "mute",
  ...
}
```

前端通过 `data-tone` 属性切换 CSS 自定义属性：

```css
[data-tone="mute"] {
  --tone-bg: #0c0b0a;
  --tone-bg2: #1c1a18;
  --tone-accent: #6b635a;
  --tone-text: #a09890;
  --tone-bright: #d0c8c0;
}
```

JS 在进入人生/结局页时设置：

```javascript
const tone = char.tone || 'ember';
document.getElementById('screen-life').setAttribute('data-tone', tone);
document.getElementById('screen-ending').setAttribute('data-tone', tone);
```

### CSS 变量与 Tone 的关系

收藏页使用的是**全局暗色调配色**（`--bg`, `--text` 等），不使用各 Tone 的独立配色。Tone 配色（`--tone-bg`, `--tone-accent` 等）仅在**人生播放页**和**结局页**生效。

收藏页的设计原则：统一暗底 + 最小装饰，不按每个 Tone 的颜色渲染卡片——避免视觉过于花哨，保持古卷的沉静感。

### 常见错误

- **不要**在收藏页按 Tone 颜色给卡片上色（如 blaze 的卡片用赭红边框）——这会打破统一的视觉节奏
- **不要**把 Tone 的英文名直接显示给用户——用户看到的永远是中文标签
- **不要**用圆角药丸按钮做 Tone 切换——与古籍气质冲突，使用下划线指示器
